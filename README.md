# 前言

欢迎来到基于SSM的研究生管理系统项目！该项目旨在为研究生院提供一种高效、便捷的管理方式，通过此系统，管理员和教师可以轻松管理研究生的相关信息，提高工作效率。以下是该项目的详细介绍。

# 内容介绍

本项目主要包含以下功能模块：学生信息管理、课程管理、成绩管理、导师管理等。系统采用前后端分离的设计模式，前端负责展示界面，后端处理业务逻辑和数据处理。通过使用Spring、SpringMVC和MyBatis等框架，确保了系统的稳定性和可扩展性。同时，前端采用了Vue、JS和CSS3等技术，提升了用户体验。

# 技术介绍

- 语言：Java
- 使用框架：Spring、SpringMVC，MyBatis
- 前端技术：JS、Vue、CSS3
- 开发工具：IDEA/Eclipse
- 数据库：MySQL 5.7/8.0
- 数据库管理工具：phpstudy/Navicat
- JDK版本：jdk1.8
- Maven：apache-maven 3.8.1-bin
- 前端环境：Node.Js 12\14\16
